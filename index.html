Projects

PROJECT 01
Heart Disease Risk Analyzer
Classical ML pipeline predicting heart disease risk from patient measurements. Combined 4 UCI datasets (920 patients), compared Logistic Regression, Random Forest and XGBoost with honest cross-validation evaluation. Deployed as a live web app.
79.1% CV Accuracy Python - Scikit-learn - XGBoost - Streamlit

PROJECT 02
Pneumonia Detection from Chest X-rays
Deep learning model using transfer learning on ResNet18 to detect pneumonia in chest X-rays. Tackled class imbalance with data augmentation and class weights, improving normal case accuracy from 44% to 84%.
92.5% Accuracy Python · PyTorch · ResNet18 · Hugging Face
GitHub ↗ Live Demo ↗
PROJECT 03
ECG Arrhythmia Detector
Signal processing pipeline detecting abnormal heartbeats in ECG recordings. Filters raw signals, detects R peaks, extracts beat segments from 48 patient recordings (102,251 beats) and classifies arrhythmias with Random Forest.
98.7% Accuracy Python · neurokit2 · wfdb · Scikit-learn
